1. Pengenalan Struktur File HTML5

HTML dari Nol - 10:24

HTML (HyperText Markup Language) adalah kerangka dasar dari setiap website. Berfungsi ibarat "tulang" manusia, HTML memberitahu browser bagaimana cara menampilkan struktur halaman dengan mendefinisikan elemen-elemen seperti paragraf, judul, gambar, dan formulir.

Struktur Wajib File HTML

Setiap file HTML yang valid menurut standar HTML5 harus memiliki deklarasi `DOCTYPE` beserta blok utama `html`, `head`, dan `body`.

/* Contoh struktur kerangka HTML5 murni */
<!DOCTYPE html>
<html lang="id">
<head>
<!-- Meta data tidak terlihat di layar user -->
<title>Halaman Pertama</title>
</head>
<body>
<!-- Konten yang akan muncul di layar -->
<h1>Halo Calon Developer!</h1>
</body>
</html>

Penugasan Praktek

Persiapkan editor di komputermu sendiri!

Buka teks editor favoritmu (misal: Visual Studio Code, Sublime Text, atau sekadar Notepad) dan ikutilah instruksi di bawah ini:

  1. Buat sebuah file baru dan simpan dengan nama tugasku.html
  2. Ketik ulang (jangan menyalin) struktur html dasar yang telah dijelaskan di materi atas.
  3. Isi bagian <title> dengan kalimat "Web Pertamamu".
  4. Buka file tersebut menggunakan browser (klik dua kali pada file tugasku.html), lalu lihat tab atas browser-mu! Anda akan melihat judul yang Anda tulis.
